正式运行前怎么填默认值

步骤动作怎么做注意事项
1先复制模板保留 default-values.template.json 不动，正式运行前复制为 default-values.local.json。本地文件可改，外发模板仍保持干净。
2填写 Excel打开 A8默认值配置说明.xlsx，在“默认值总表”和各流程 sheet 的“建议修改为”列填写现场默认值。只填业务默认，不填账号、密码、验证码、草稿 ID、真实项目材料。
3确认启用边界把“是否启用”列逐项确认；没有把握的字段保持空值或标记为否。宁可让运行前校验告警，也不要为了省事硬填。
4同步到本地 JSON把确认后的默认值同步到 config/default-values.local.json，正式运行前由 Codex 或操作者合并到本次输入 JSON。local 文件建议不进入外发包、不提交、不分享。
5运行前预检正式跑 Skill 前先检查必填项、模式边界和默认值来源。前三个流程可 fast；存货批量只允许 normal/debug。
6按优先级合并用户当前明确值 > 用户材料值 > default-values.local.json > default-values.template.json > 空值并告警。同一字段有冲突时，以用户当前明确值为准。
7缺关键值阻断项目名、客户、人员、金额、料号、数量、单价等关键字段缺失时不保存。可默认字段才自动补；不可默认字段必须让人确认。
推荐设计：模板默认值和现场本地默认值分离。外发包只带 default-values.template.json，正式运行前由使用方生成或填写 default-values.local.json。
