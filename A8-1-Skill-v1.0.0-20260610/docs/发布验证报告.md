## 默认值产品边界

v1.0.0 不把 `default-values.template.json` 作为脚本强制自动读取的运行时配置源。正式运行前的逻辑是：

1. 使用方在 `config/A8默认值配置说明.xlsx` 中确认默认值。
2. 使用方复制并维护本地 `config/default-values.local.json`。
3. Codex 或操作者在运行前按“用户明确值 > 用户材料 > local 默认值 > template 默认值 > 空值告警”合并。
4. 合并后的完整输入写入本次 `A8_ASSISTANT_INPUT_JSON_FILE`。

## 风险边界

- 如现场 A8 字段、控件名称、表单模板或待发列表结构变化，仍需进入 `normal` / `debug` 模式复核。
- 集成项目创建单无基线兜底依赖已验证的默认客户/人员映射；如用户输入新的客户或人员，需先扩展映射或进入人工复核。
- 集成项目立项申请全量材料中存在 A8 物料选择器无法匹配的料号；当前策略是保留人工复核告警并继续保存待发。
- `default-values.local.json`、账号密码、验证码、A8 草稿 ID、真实项目材料和运行产物不得进入外发包。
